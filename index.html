TRACKS

TRACK 1

Plant disease detection

Develop a machine learning model that can accurately identify and segment regions of plant leaves affected by disease/pest in images provided in a dataset. The model should be able to classify the images into different disease categories and highlight the affected areas on the leaves. The participants will be judged based on the accuracy, inference time and novelty of the architecture used.

TRACK 2

Integrating mobile applications and cloud based expert systems

Develop an Android mobile application that can interact with a provided REST API for plant disease detection. The app should be able to send an image of a plant leaf to the API, receive the disease classification, disease stage, and other metadata in response, and display the results in a user-friendly interface. The participants will be judged on the app's User Interface, User Experience and functionality of the API integration.

TRACK 3

Path planning (RGB data, depth data)

Develop a computer vision algorithm/machine learning model that can accurately identify drivable areas in an unlabelled dataset of RGB and depth data from a sugar cane field. The data is taken from a restricted area where robots are used to move around. The algorithm should be able to distinguish the drivable areas from non-drivable areas in the field. The participants will be evaluated based on the accuracy of the algorithm and its ability to generalize to new data.
